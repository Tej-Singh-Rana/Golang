# fmt 

#### Go Docs

- [fmt Package](https://golang.org/pkg/fmt/)

> Import the `fmt` package.

```code
package main

import "fmt"
```
> Imported the multiple packages such as `fmt` and `os`.

```code
package main

import (
   "fmt"
   "os"
 )
```

#### Command Line 

- Command line tools to get more help.
 
> To read the documentation about "fmt" package.

```cmd
$ go doc fmt
```

> For the help

```cmd
$ go help fmt
```
